<p>在這時代裡，許多基督徒反對十一奉獻。<br/>
結果，喊十一奉獻最大聲的，反而常常是那些靈恩派與成功神學的陣營。<br/>
偏偏，這些靈恩派與成功神學教導的相關奉獻觀念，大量都是錯誤的。<br/>
身為基督徒，我們必須持守純正信仰，不偏左偏右。<br/>
而ICCP信仰宣言，重申支持十一奉獻，正如我們園地一貫的教導。<br/>
<!--more--><br/>
<br/>
<br/>
ICCP信仰宣言 主題十五 關於合乎聖經的經濟制度<br/>
<br/>
<br/>
第十六條<br/>
我們確認十一原則（就是10%）是一條合乎聖經的基本經濟原則。<br/>
我們否認十一原則對今天的基督徒不適用；或者一個基督徒沒奉行十一奉獻，仍可達成他的經濟職責。<br/>
<br/>
<br/>
創14：18-20，利27：30-32，瑪3：8-12，太23：23，來7：9-10<br/>
<br/>
<br/>
<br/>
<br/>
說明：<br/>
ICCP信仰宣言的重要性，是因為這是為當今教會教導已經大量混亂的時代，重申二千年正統歷史傳承的信仰，讓大家知道什麼是對、什麼是錯。<br/>
並且，這是經過全世界大量信仰純正的神學家所檢驗過，確認這是合乎全本聖經的正確教義與教導。<br/>
因此，假使我們自己觀念和ICCP相反時，我們最好謙卑下來，重新檢驗自己觀念，好好查考聖經，以便回歸正統純正的信仰內容。<br/>
<br/>
<br/>
關於十一奉獻的觀念，我們園地已經有過很多篇文章，就不再重複。<br/>
大家可以自行閱讀：<br/>
『十一、奉獻、濟貧、教會金錢管理相關文章』<br/>
<a href="/posts/269196064">/posts/269196064</a><br/>
<br/>
<br/>
簡單說，十一奉獻的正確觀念是：<br/>
1. 今日信徒一樣要守十一奉獻。<br/>
2. 不管自己是不是富人或窮人，只要有收入，一樣都要守十一奉獻。窮人並沒因此就可以不用守。<br/>
3. 十一奉獻的計算方式，屬於個人自由，依信心與領受而定即可，不須拘泥。<br/>
4. 守十一奉獻會被祝福，沒守十一奉獻不會被咒詛，但得不到祝福。<br/>
5. 十一奉獻無關稱義領域，而是屬於成聖領域。<br/>
6. 現今做不到，努力便是，不得因為很難做或現在做不到，就反對十一奉獻。<br/>
7. 十一奉獻並不是只能獻在自己教會。<br/>
8. 十一奉獻的精義，起碼有：<br/>
a. 給聖工人員養生（含福音機構）<br/>
b. 為聖工用途<br/>
c. 濟助貧困<br/>
9. 不得因為有人教導錯誤的十一奉獻觀念，就否定十一奉獻。<br/>
10. 不得因為有人有錯誤的十一奉獻行為，就否定十一奉獻。<br/>
<br/>
