<p>事實上，聖經雖然常提到『利未人無份無業』，但整個情形非常複雜，不是那麼簡單。<!--more-->基本上，利未人並沒有分到地業。<br/>
『惟獨利未人要辦會幕的事、、、、他們在以色列人中不可有產業』（民18：23）<br/>
『利未人在他弟兄中無分無業，耶和華是他的產業』（申10：9）<br/>
『只是利未支派，摩西沒有把產業分給他們』（書13：33）<br/>
<br/>
<br/>
當初進迦南地時，各支派開始要分地分產業。<br/>
在約書亞記14章至19章，詳細列舉了各支派分得的地業。<br/>
但不管是摩西生前，還是約書亞分地時，都特別強調，利未人是無份無業，不參與各支派的土地產業分配的。<br/>
<br/>
<br/>
但是，我們要小心解讀所謂的『無份無業』，<br/>
因為，以下經文就顯示出沒那麼簡單：<br/>
『沒有把地分給利未人，但給他們城邑居住，並城邑的郊野，可以牧養他們的牲畜，安置他們的財物』（書14：4）<br/>
『你吩咐以色列人，要從所得為業的地中把些城給利未人居住，也要把這城四圍的郊野給利未人。這城邑要歸他們居住，城邑的郊野可以牧養他們的牛羊和各樣的牲畜，又可以安置他們的財物』（民35：2-3）<br/>
在這裡，我們可以看到，利未人分到：<br/>
1.城（當然就有房子）<br/>
2.城的郊野（當然就有地）<br/>
同時，這經文也顯示出，利未人不僅有某些城、城郊土地，而且還有<br/>
3.私人牛羊<br/>
4.還有私人財產。<br/>
<br/>
<br/>
『於是以色列人照耶和華所吩咐的，從自己的地業中，將以下所記的城邑和城邑的郊野給了利未人』（書21：3）<br/>
各支派在分得地業之後，還要從自己地業中，將一些城分給利未人。<br/>
約書亞記第二十一章就詳細記錄這件事。<br/>
<br/>
<br/>
利未人雖然在支派大型土地產業分配裡，不參與分配，<br/>
但其實，他們是有一些城、城郊、牛羊、私有產業的。<br/>
不僅如此，上帝還規定：<br/>
1. 上帝就是利未人的產業<br/>
2. 各支派的十一奉獻，就是利未人的產業<br/>
『凡以色列中出產的十分之一，我已賜給利未的子孫為業；因他們所辦的是會幕的事，所以賜給他們為酬他們的勞』（民18：21）<br/>
<br/>
<br/>
所以，任何以現今牧師有些是有地有業，甚至有其他收入，就否定我們該奉獻給牧師，這是錯誤解釋『利未人無份無業』的經文，並且違反全本聖經『傳福音的靠福音養生』的經文的總原則的。<br/>
<br/>
<br/>
當然，『耶和華就是利未人的產業』，這觀念，還有很多精義在其中，但因不是本篇主題，所以不再多講。<br/>
但是，現今真的有不少牧師，早就忘記身為神職人員，『上帝』才是神職人員最大的產業。<br/>
你盡忠職守、不貪愛錢財、不在乎名利富貴、不在乎教會待遇福利高低、、、、，上帝就會負責祂對你的應許，當你終生的產業。<br/>
那些看重薪水福利多寡、看錢很重、拼命教導信徒十一奉獻只能往自己教會獻的牧師，恐怕早已忘記上帝才是利未人最大的產業。<br/>
<br/>
<br/>
另外，十一奉獻並不是只單給利未人使用，<br/>
因為，十一奉獻起碼可以用在這些人與用途上的：<br/>
1. 聖工人員<br/>
『凡以色列中出產的十分之一，我已賜給利未的子孫為業；因他們所辦的是會幕的事，所以賜給他們為酬他們的勞』（民18：21）<br/>
2. 濟貧<br/>
『每逢三年，就是十分取一之年，你取完了一切土產的十分之一，要分給利未人和寄居的，與孤兒寡婦，使他們在你城中可以吃得飽足』（申26：12）<br/>
而利未人，可不是只有自己教會、自己牧師牧師才算，而是所有從事聖工的教會、機構，所有從事聖工的人員，如：牧師、傳道、幹事、行政人員、、、都算。<br/>
<br/>
<br/>
因此，當牧師行為不良時，我們依然有很多標的可以進行十一奉獻的重分配，但不能因為牧師行為不良，就廢棄給上帝的十一奉獻。<br/>
<br/>
<br/>
<br/>
<br/>
小小羊</p>
