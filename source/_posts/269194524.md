<p>『要獻得可蒙悅納』（利19：5）<br/>
『你們不可獻上，因為這不蒙悅納』（利22：20）<br/>
<!--more--><br/>
<br/>
<br/>
<br/>
太多人已經以為，只要為了傳福音，即使犯罪行惡也無所謂、違反聖經也無所謂。<br/>
也太多人以為，只要有愛心、一切包容便是，教會裡有任何假道、任何罪行，反正都無所謂，都是做聖工。<br/>
<br/>
<br/>
問題是，上帝真的喜悅這樣的事？<br/>
<br/>
<br/>
要探討這部分，有很多角度可以切入。<br/>
我們今天只用一個很少被講到的角度來提醒：<br/>
『獻得蒙悅納』vs『獻得不蒙悅納』。<br/>
<br/>
<br/>
獻祭時，是要獻給上帝的。<br/>
獻給上帝的東西，因個人能力、財力有別，上帝並沒全部都要求大家一定都要獻出同樣的東西。<br/>
問題是，上帝要我們獻好的給祂，不要我們用隨便的心態來獻。<br/>
<br/>
<br/>
有些信徒，把家中不要的東西拿到教會去，美其名叫「奉獻給主」。<br/>
這真的是奉獻？<br/>
還是將教會當垃圾場，然後又希望博取美名？<br/>
我不是說不能將二手的東西獻給主、奉獻到教會來。<br/>
而是我們真的要很謹慎小心自己內心的動機，真的敢對上帝說：<br/>
「主啊，我將這好東西獻給你」？<br/>
<br/>
<br/>
美國有一個很感人的真實小故事：<br/>
一個小女孩，家裡很窮，住在破爛的地方，也沒甚麼玩具和學用品。<br/>
她很希望去念主日學，但因教會教室容量很小，有時她無法進去，只能坐在教會外面哭泣。<br/>
她的生命不長，小小年紀就生病死了。（在經濟貧困的區域，有多少小孩能健康長大？）<br/>
死後，牧師去她家探訪，在她非常寒酸破爛的個人用品中，發現一個從垃圾堆檢來的破爛小錢包，裡面有一張小紙條，還有五十七分錢（連一塊錢都不到）。<br/>
紙條寫著：<br/>
「這是獻給上帝的，要讓我們小小的教會能建得大一點，使更多的小朋友都能上主日學」<br/>
<br/>
<br/>
當牧師在講台上，講出這件事，唸出那段小紙條的內容時，<br/>
從此之後，奉獻大量湧進，那些奉獻讓他們蓋了一間大教堂、容納數千人的大禮堂、一間醫院、還有一棟主日學大樓。<br/>
<br/>
<br/>
破爛的東西，誰說不能拿來奉獻？<br/>
貧賤的東西，誰說不能拿來奉獻？<br/>
窮寡婦奉獻兩個小錢，蒙主悅納（可12：42）；<br/>
小朋友奉獻五餅二魚的晚餐便當，蒙主悅納（太14：17）<br/>
但是，請注意再注意我的提醒：<br/>
你真的是獻出最好的了嗎？<br/>
假使你有能力，可是卻拿爛東西出來獻，還敢講自己奉獻給主，請小心獻得不蒙悅納！<br/>
<br/>
<br/>
『蒙悅納的，不是自己稱許的，乃是主所稱許的』（林後10：18）<br/>
蒙不蒙悅納，不是自己自欺欺人、自欺欺上帝就行得通的。<br/>
不要以為自己所獻的，就是了不起，就一定蒙上帝悅納。<br/>
沒這回事！<br/>
蒙不蒙悅納，是要由上帝決定，不是我們自己可以擅自用私意決定的。<br/>
<br/>
<br/>
當我們奉獻，當我們從事聖工，請千萬小心，不要以為自己有奉獻、做聖工，上帝就悅納。<br/>
當我們不遵照上帝規定來奉獻、不照上帝規定來做聖工，即使做再多，會蒙悅納嗎？<br/>
輕則所做皆成空，重則被上帝擊殺而亡。<br/>
<br/>
<br/>
『若有人用金、銀、寶石、草木，禾秸在這根基上建造，各人的工程必然顯露，因為那日子要將它表明出來，有火發現；這火要試驗各人的工程怎樣。人在那根基上所建造的工程若存得住，他就要得賞賜。人的工程若被燒了，他就要受虧損，自己卻要得救；雖然得救，乃像從火裡經過的一樣』（林前3：12-15）<br/>
不管是對自己靈魂的工程，還是對團體的聖工，都是一樣的道理。<br/>
<br/>
<br/>
『亞倫的兒子拿答、亞比戶各拿自己的香爐，盛上火，加上香，在耶和華面前獻上凡火，是耶和華沒有吩咐他們的，就有火從耶和華面前出來，把他們燒滅，他們就死在耶和華面前』（利10：1-2）<br/>
『到了拿艮的禾場，因為牛失前蹄，烏撒就伸手扶住神的約櫃。神耶和華向烏撒發怒，因這錯誤擊殺他，他就死在神的約櫃旁』（撒下6：6-7）<br/>
貴為大祭司的兒子、未來的接班人；<br/>
善意如烏撒出於好心；<br/>
上帝一樣擊殺！<br/>
請不要以為可以隨便用不合聖經的東西來從事聖工、拿我心裡感覺的善意來幫助上帝。<br/>
請千萬不要搞錯了！<br/>
上帝要人用祂喜悅的方式來服事祂，不是用我們自己想出來不合聖經的方式來服事祂。<br/>
<br/>
<br/>
當然，上帝並沒叫我們當死板版、甚麼創意都沒有的人。<br/>
像製作會幕、聖衣、各種聖工的工匠比撒列，上帝可是聖靈充滿他，讓他有智慧可以做出、想出各種精巧的好作品。<br/>
『我也以我的靈充滿了他（比撒列），使他有智慧，有聰明，有知識，能做各樣的工，能想出巧工，用金、銀、銅製造各物，又能刻寶石，可以鑲嵌，能雕刻木頭，能做各樣的工』（出31：3-5）<br/>
問題是，在發揮創意前，必需先謹守上帝的規定：<br/>
『要謹慎做這些物件，都要照著在山上指示你的樣式』（出25：40）<br/>
這種『照著上帝指示』的描述，在製作各種聖工用品時，大量出現在經文裡，可見上帝對這部分的看重。<br/>
<br/>
<br/>
換言之，必需『先』『符合上帝規定』的前提，『之後』才能在這根基上『進行創意發揮』；<br/>
不能顛倒過來，先發揮創意，然後才去聖經硬找經文來附和；<br/>
甚至連經文都不找，反正用『聖靈已經啟示我』這類主觀經歷來為自己行為護航。<br/>
<br/>
<br/>
知道為什麼上帝對以色列人在製作會幕、聖衣、燈臺、約櫃、各種器具、、、、，會先有詳細規定嗎？<br/>
原因之一就是：<br/>
假使不這樣規定，反而完全讓人自由發揮，有無可能變成用腦袋去想出偶像崇拜，然後用這樣的方式來製作這些東西？<br/>
<br/>
<br/>
舊約以色列人愛拜偶像，事蹟太多了，不用多題。<br/>
在此，我要提醒一個很可怕的例子：<br/>
連神聖的東西，都可以被拿來當成褻瀆神的工具。<br/>
摩西製作了銅蛇，只要仰望銅蛇，被蛇咬的就好了。<br/>
『摩西便製造一條銅蛇，掛在杆子上；凡被蛇咬的，一望這銅蛇就活了』（民21：9）<br/>
這個救人的銅蛇，被以色列人辛辛苦苦保存，從摩西出埃及記開始，他們就歷盡千辛萬苦保存攜帶，連士師記長期被異族欺凌的漫長年歲都沒有破壞，還一直保存到大衛、所羅門輝煌的王朝之後，繼續保管下來，歷經南北分裂、諸王戰亂，還沒有弄壞。<br/>
問題是，也是這寶貴的銅蛇，竟然也是被以色列人當成偶像來膜拜、燒香獻祭，直到被希西家王毀滅為止。<br/>
『他廢去邱壇，毀壞柱像，砍下木偶，打碎摩西所造的銅蛇，因為到那時以色列人仍向銅蛇燒香。希西家叫銅蛇為銅塊（或作人稱銅蛇為銅像）』（王下18：4）<br/>
連當初是正確敬拜神的好東西，都可以被拿來當錯誤的偶像膜拜，引發上帝的憤怒，<br/>
更何況連聖經依據都沒有、錯誤越過聖經的敬拜與服事，上帝會悅納嗎？<br/>
<br/>
<br/>
上帝不箝制人的自由，但祂要人願意順服在祂面前，在祂規定的根基上，發揮自由與創意，這樣才能蒙神悅納。<br/>
否則的話，按照人類墮落的理性，一旦沒有事先的規範與限制，真的很容易就用偶像崇拜的方式來服事神，而且還以為這樣很好。<br/>
問題是，現今有多少人有按照聖經來從事聖工的觀念？<br/>
有多少人對傳福音會小心謹慎自己所行是否越過聖經、違反聖經、甚至傳出錯誤的假福音？<br/>
<br/>
<br/>
我們已經犯大罪了，還沒有警覺！<br/>
<br/>
<br/>
願上帝憐憫我們！<br/>
<br/>
<br/>
小小羊</p>
