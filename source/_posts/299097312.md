<p>問題：我已讀過你寫的關於討價還價的禱告的批判。<br/>
但是我還是有疑惑，討價還價的禱告，真的不可以嗎？、、、、<br/>
（隱私均去除）</p>
<p> </p>
<p>答：<br/>
請多比較我提供的那篇『對上帝生氣』，和『討價還價的禱告』二文有什麼不同。 <br/>
請特別注意他們引用的聖經例子與生活實例，比較其中的差異。 </p>
<p><br/>

<p>生上帝的氣<br/>
<a href="http://www.chinesebiblicalcounseling.net/suffering/angry_at_God.htm" target="_blank">http://www.chinesebiblicalcounseling.net/suffering/angry_at_God.htm</a></p>
<p><br/>
事實上，題目定成肯定句的『討價還價的禱告』，就已經是錯誤的，<br/>
只是定題目的人自己可能都不知道而已。 <br/>
因為，上帝本來就不能讓人討價還價的。 </p>
<p><br/>
誇張的題目，當然可以吸引人。 <br/>
但是，我們要很小心，要看看誇張所要表達的內容，是否是正確的觀念。 </p>
<p><br/>
題目訂為『大善人耶穌』、『酷炫老師耶穌』，這都可以； <br/>
題目訂為『與檳榔西施打情罵俏的耶穌』絕對不行。 <br/>
我們不能因為耶穌和妓女談話、關心妓女，<br/>
就塑造出一個會與女人打情罵俏的耶穌形象。 <br/>
對於上帝的『敬畏』，我們絕對不能忘記。 </p>
<p><br/>
人對神，可以『親密』，但不能『沒大沒小』（喪失分寸）。 <br/>
連變成人的神耶穌基督，與祂的門徒之間，<br/>
我們也看不見那種『沒大沒小』的範例。 </p>
<p><br/>
同樣的，人絕對可以向上帝『祈求』，但不能對上帝『討價還價』；<br/>
同樣的，『人對上帝發脾氣』是『軟弱』，上帝通常會『包容』，<br/>
但不能認為這是『值得學習的好禱告』。 </p>
<p><br/>
在聖經裡，人的軟弱被上帝包容的例子多得很，<br/>
但把這種上帝『寬容』當成上帝『悅納』、認為值得效法學習，這就差遠了。 <br/>
請留意『對上帝生氣』那篇文章，文章內容是『人不能對上帝生氣』。 <br/>
而且，請注意文章中想到上帝後開始對上帝生氣的那個例子，<br/>
文章是如何來評論這樣的行為的？ <br/>
那個評論就是正確的： <br/>
「小董為甚麼不抱怨上帝呢？因為他根本沒想到上帝。<br/>
後來他想到上帝了，也就生上帝的氣。 <br/>
小董算不算進步了？ 當小董想到上帝，這是邁向正確的第一步。<br/>
但是，他必須繼續朝正確的方向走。」 </p>
<p><br/>
『但是，他必須繼續朝正確的方向走』！就是這句話！<br/>
差了這句話，就差十萬八千里。 <br/>
在楊腓力該文中，就是缺乏這樣的觀念。 <br/>
假使他認為人在對上帝大吵大鬧摔東西之後，要悔改認罪、追求更正確的路，<br/>
那就是對的。 <br/>
但很不幸的，你在他文章裡，有讀到這樣的教導嗎？ </p>
<p><br/>
約拿書是最明顯的例子： <br/>
上帝要我們效法約拿的行為，因為上帝認為他的行為很好？是這樣嗎？ <br/>
上帝對約拿的行為，是『寬容』，還是認為這是『值得效法』？ </p>
<p><br/>
為什麼這問題很重要？ <br/>
因為那牽涉到我們要如何『輔導受苦者』的問題。 </p>
<p><br/>
1.請告訴我，根據楊腓力那篇文章的論點，<br/>
現在假使你遇到一個『受苦者』，他問你： <br/>
「我可不可以對上帝大吵大鬧摔東西？」 <br/>
請根據楊腓力的論點告訴我，你如何回答這人？ </p>
<p><br/>
2.同時，也請告訴我，那個被大吵大鬧摔東西的傳道人的禱告吵到的『鄰居』，<br/>
他來問你： <br/>
「我隔壁可以這樣大吵大鬧摔東西吵到我嗎？<br/>
他可以這樣禱告嗎？<br/>
我可不可以也這樣大吵大鬧摔東西禱告？」 <br/>
請根據楊腓力的論點告訴我，你如何回答他？ </p>
<p><br/>
3.再來，請告訴我當你遇到一個被先生大吵大鬧摔東西禱告嚇到哭的妻子，<br/>
當她問你： <br/>
「我先生可以這樣禱告嗎？<br/>
我可以學他一樣禱告嗎？<br/>
我可以教我的孩子學那樣的禱告嗎？ 」</p>
<p><br/>
這三個問題很重要，你一定要回答我。 <br/>
否則你可能還無法很精準看見我深深擔憂的。 <br/>
當我看見我所尊敬的一些傳道人，已經贊同楊腓力的論點時，<br/>
我就發現事態嚴重了，因為連這些好的傳道人都沒看出問題點。 </p>
<p><br/>
假使你願意的話，不妨嘗試用楊腓力的論點，以及我附的那篇文章的論點，<br/>
來回答這三個實際上真的可能遇到的問題。 <br/>
你應該會開始發現那裡面的重大差異點。</p>
<p> </p>
<p>你認為人可對上帝發洩情緒，<br/>
事實上，這種『人可以對上帝發洩情緒』的觀念，絕不是你獨有而已，<br/>
它一定是很普遍的觀念，而且是很多好基督徒都認為這是正確的觀念。 </p>
<p><br/>
我擔心而且懷疑的是： <br/>
可能你們從來沒有真正聽過這部分正確的教導。 <br/>
假使你們過去並沒聽過『人不能對上帝發洩情緒』的教導，<br/>
反而聽的都是『人可以對上帝發洩情緒』的教導，<br/>
你們當然無法知道正確的是什麼。 </p>
<p><br/>
因此，我要先確認一次你真正的意思。 <br/>
你認為的人可以對上帝『發洩情緒』，<br/>
是指『人可以朝上帝發洩對祂的不滿意嗎』？ <br/>
我不是說告訴上帝我不滿意『人類』，而是指特別針對『上帝本身』。 <br/>
（不是告訴上帝『我對某人不爽』，而是告訴上帝『我對上帝你不爽』）</p>
<p><br/>
我換個方式講好了，你所謂的對上帝發洩情緒，是指哪一種情形？ <br/>
1.我今天和人吵架，然後我對上帝說出我『對那人』的憤怒或不滿。 <br/>
2.我今天和人吵架，然後我對上帝說出我『對上帝』的憤怒或不滿。 <br/>
請特別注意都是對上帝禱告（態度強弱沒關係），但重點是『攻擊的對象不同』。 <br/>
你所謂的對上帝發洩情緒，是指哪一種？ </p>
<p><br/>
假使是前者，是對上帝表達出對『那人』的不滿，<br/>
那這種對上帝發洩情緒是可以的。</p>
<p>假使是後者，是對上帝表達出對『上帝』的不滿，<br/>
那這種對上帝發洩情緒是不行的。</p>
<p><br/>
楊腓力的論點，<br/>
就是很標準的『人可以對上帝發洩情緒』、『人可以對上帝生氣』；<br/>
相反的，我提供的那篇聖經輔導的文章，<br/>
就是很標準的『人不可以對上帝發洩情緒』、『人不可以對上帝生氣』。 </p>
<p><br/>
除非我們先解決這個基本預設，就是人到底可不可以對上帝發洩情緒，<br/>
否則的話，根本問題沒解決，後面的會很難處理。 <br/>
接下來就更別提真正遇到這樣的輔導問題時，我們必需採取什麼立場了。 </p>
<p> </p>
<p>我再強調一次，這不是譴責，而是利用這樣的機會，<br/>
我才能比較深入去抓住你問題的核心，然後幫助你更清楚這些觀念。 <br/>
假使連你這樣的好基督徒都沒聽過這些基本觀念，其他的更甭講。 <br/>
這，就是我深深擔憂之處，也是為何我對楊腓力那篇會很憤怒的原因之一。</p>
<p><br/>
我們當然可以對上帝發洩『情緒』，<br/>
但是，那是針對『人』，不能針對『神』。<br/>
我可以對上帝發洩我對『某人』的憤怒，<br/>
但我不能對上帝發洩我對『上帝』的憤怒。</p>
<p><br/>
可是，楊腓力之類那些教導『人可以對上帝發洩情緒』的，<br/>
並不是教導大家可以對上帝發洩我們對『某人』憤怒的情緒，<br/>
而是教導大家可以對上帝發洩我們對『上帝』憤怒的情緒。</p>
<p><br/>
問題是，詩篇大量的咒詛詩，都是在對上帝發洩我們對『某人』的憤怒；<br/>
但是，聖經並沒要我們發洩對『上帝』的憤怒。</p>
