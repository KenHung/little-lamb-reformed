這是基督教小小羊園地的 Responsive 版本，使用了 GitHub Pages, Jekyll, Vue, Algolia。

GitHub Actions 會自動從基督教小小羊園地的 API 下載並儲存最新文章，然後更新網站。所有文章以 Markdown 格式儲存，Jekyll 會將其轉換為網頁，再上載至 GitHub Pages 上。
