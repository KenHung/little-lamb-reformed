title: 教會牧師是CEO，還是牧羊人？
last_modified_at: 1633766934

<p>教會牧師是上市公司總裁、專業經理人、執行長這類人士嗎？<br/>
<br/>
<!--more-->很抱歉，我將這些職稱，全部都用很不嚴謹的方式稱為CEO，要不然文章會變得又臭又長。<br/>
什麼是我這裡指的CEO呢？<br/>
就是那種領薪水，有很高的權力進行決策與管理的公司領導人。<br/>
這些人與公司的結合，基本上是一種利益的結合：<br/>
公司出錢，CEO出人。<br/>
<br/>
<br/>
對CEO而言，他在乎的是：你們公司要付我多少薪水、給我多少福利；<br/>
對公司而言，他在乎的是：你這個CEO能帶給我們多少利潤。<br/>
<br/>
<br/>
對CEO而言，這家公司只是他眾多選擇的其中之一而已，隨時可以屁股拍拍走人。<br/>
對CEO而言，他只需對董事會之類負責，而董事會通常不太會插手公司經營，因此CEO手中權柄極大。<br/>
在公司裡，CEO就是老大，而且非常的大，沒人可管理他。<br/>
<br/>
<br/>
通常CEO是在辦公室吹冷氣的人，不是在外面拋頭顱撒熱血、吹風淋雨的人。<br/>
畢竟是大將軍，重點在決策與發號施令，然後交給轄下各個主管與小兵兵們去執行。<br/>
CEO並不是第一線執行的人，畢竟大將軍去當小兵兵對公司並不太合適。<br/>
公司聘請CEO是要他的腦袋的，不是要他的第一線作業能力的。<br/>
第一線作業員的薪水低多了，付給CEO的高薪資，不該是讓他去當第一線作業員用的。<br/>
<br/>
<br/>
以上所說的，絕無對CEO不敬的意思。<br/>
事實上，在企業裡，CEO是很重要的，而且權利義務真的是很清楚的。<br/>
對公司而言，付出高薪給CEO、給予高權柄，只要能為公司帶來利潤，這是值得的。<br/>
<br/>
<br/>
但是，教會牧師是CEO嗎？<br/>
<br/>
<br/>
現今已經有多少牧師，已經以為自己是CEO了？<br/>
<br/>
<br/>
1. 薪水多多益善，福利越多越好，看重的是能享有多少權益。<br/>
2. 權柄越高越好，越少人管我越佳，重點是發號施令、你們去做。<br/>
<br/>
<br/>
對這些牧師而言，我想講一個觀念：<br/>
你們既然想當CEO，那麼，大家就該照民間企業的遊戲規則來玩。<br/>
所以，教會付你多少薪水，你就該達成多少業績，否則就減薪或走人，而且雙方都該白紙黑字簽下合約，定好權利義務。<br/>
<br/>
<br/>
『教會付你多少薪水，你就該達成多少業績』。<br/>
想當CEO？<br/>
不要只記得第一句，卻忘記第二句。<br/>
<br/>
<br/>
問題是，要講薪水這類權利，大家都要；<br/>
講到業績這類義務，大家閃邊？<br/>
<br/>
<br/>
多少牧師，是抱著CEO心態來牧會的？<br/>
不！<br/>
這還太客氣了，因為CEO還知道有達成業績的壓力，而這些想當CEO的牧師，卻只想享有民間企業CEO的高待遇與高權柄，但不會想達成企業要求的高業績。<br/>
<br/>
<br/>
假使，教會牧師是CEO，那會是教會的悲哀，更是腐爛的基督的身體。<br/>
『教會是他的身體』（弗1：23）<br/>
<br/>
<br/>
為什麼？<br/>
因為基督是牧羊人，不是CEO。<br/>
<br/>
<br/>
有差嗎？<br/>
有！<br/>
牧羊人為羊捨命，遇危險身先士卒；<br/>
雇工拿錢辦事，隨時可落跑。<br/>
<br/>
<br/>
『我是好牧人；好牧人為羊捨命。<br/>
若是雇工，不是牧人，羊也不是他自己的，他看見狼來，就撇下羊逃走；<br/>
狼抓住羊，趕散了羊群。雇工逃走，因他是雇工，並不顧念羊。<br/>
我是好牧人；我認識我的羊，我的羊也認識我』（約10：11-14）<br/>
<br/>
<br/>
『一個人若有一百隻羊，一隻走迷了路，你們的意思如何？<br/>
他豈不撇下這九十九隻，往山裡去找那隻迷路的羊嗎？』（太18：12）<br/>
<br/>
<br/>
牧師們，你們願意為羊捨命嗎？<br/>
牧師們，你們願意被荊棘刺傷，翻山越嶺去尋找迷失的那隻小羊嗎？<br/>
牧師們，你們認識你們的羊嗎？<br/>
知道羊的個性、恩賜、缺點、、、、？<br/>
知道要如何將他們養肥嗎？知道要如何醫治他們的傷痕嗎？<br/>
<br/>
<br/>
『人子啊，你要向以色列的牧人發預言，攻擊他們，說，主耶和華如此說：<br/>
禍哉！以色列的牧人只知牧養自己。牧人豈不當牧養群羊嗎？<br/>
你們吃脂油、穿羊毛、宰肥壯的，卻不牧養群羊。<br/>
瘦弱的，你們沒有養壯；<br/>
有病的，你們沒有醫治；<br/>
受傷的，你們沒有纏裹；<br/>
被逐的，你們沒有領回；<br/>
失喪的，你們沒有尋找；<br/>
但用強暴嚴嚴地轄制。<br/>
因無牧人，羊就分散；既分散，便作了一切野獸的食物。<br/>
我的羊在諸山間、在各高岡上流離，在全地上分散，無人去尋，無人去找。』（結34：2-6）<br/>
<br/>
<br/>
『我認識我的羊，我的羊也認識我』（約10：14）<br/>
什麼是聖經裡常講的『認識』？<br/>
我知道你叫什麼名字，你家住哪裡，家裡有幾個人，這就叫『認識』嗎？<br/>
不是的！（但很不幸的，有時，連這樣的認識，很多牧師都沒做到）<br/>
聖經裡講到『認識』，那種意思是很深遠的，是接近那種『我對你愛到從頭到腳、從內到外都深深認識你』的意思的，而且常常是用夫妻那種親密認識來類比的。<br/>
這就是上帝對人的『認識』。<br/>
主耶穌是好牧人，祂是這樣『認識』祂的羊的；<br/>
而我們今日的牧師們，又是如何認識他們的羊呢？<br/>
會友多次沒來禮拜，牧師也不會知道，也不會問候；<br/>
會友探訪，這是小組的責任、是長執的事，牧師是CEO，不用去做這種小事。<br/>
至於翻山越嶺尋找迷失的小羊？<br/>
這太辛苦、太累了，CEO不用去做。<br/>
<br/>
<br/>
『吃脂油、穿羊毛、宰肥壯的』（結34：3）<br/>
以前很多牧師是看高不看低，只牧養肥壯的羊，取他們的毛和奶；<br/>
現今很多牧師高也不屑、低也不看，肥羊還是要，但不想牧養，只想殺來吃。<br/>
牧羊人？<br/>
有多少已經變成殺羊人、吃羊人了？<br/>
『吃脂油、穿羊毛、宰肥壯的』。<br/>
<br/>
<br/>
我會不會要求太高了？<br/>
畢竟牧師也是人，不是神，<br/>
牧師只是凡人，不是耶穌基督，<br/>
所以，他們當然無法做到主耶穌那麼好，不是嗎？<br/>
<br/>
<br/>
我知道，我絕對知道，因為，我自己也是重罪的人，我也做不到很多聖經講的德行。<br/>
但是，我知道自己重罪，但還會有一絲絲的心願，願意效法主耶穌，在成聖的路上不斷努力。<br/>
然而，有多少牧師，已經忘記自己是『牧羊人』的身份了？<br/>
更甚至，我還要提醒一句話：<br/>
來當牧師，是因為要將自己獻給主，還是要混口飯吃的謀生工具？<br/>
<br/>
<br/>
對一間大型教會牧師的要求，和對中小型教會牧師的要求，當然不一樣。<br/>
我們無法要求一間五百人、一千人教會的牧師，親自去探訪每戶會友、深切認識所有會友；<br/>
但問題是，當教會只有一百人、五十人，可是牧師連會友名字都不熟、一年探訪不到一次，這是在牧什麼會？顧什麼羊？<br/>
<br/>
<br/>
是雇工，還是牧羊人？<br/>
是CEO治理教會，還是牧羊人牧養羊群？<br/>
<br/>
<br/>
CEO是找一大堆事、弄一大堆計畫給下面的人去做，是下面的人做，不是他自己做；<br/>
牧羊人是自己去做，凡事自己做，用這樣的態度當基礎來做，自己清理羊大便、自己去幫羊接生、自己去醫治病羊、自己去釘柵欄。<br/>
牧羊人不是不能找幫手，但他知道，責任是自己的、這些是自己的羊，羊沒照顧好，最該反省檢討的是自己，不是其他幫手。<br/>
幫手沒做好，牧羊人自己就要去把漏洞補起來，而不是認為事不關己，認為責任是別人的。<br/>
<br/>
<br/>
我的語氣或許很激烈，但我的心真的在淌血。<br/>
教會要復興？<br/>
靠成功神學？靠靈恩運動？靠心理學？靠各種技術？<br/>
沒錯，這確實也常可見到效果，<br/>
畢竟，安非他命、搖頭丸也可以達成效果，只是長期下來會傷身而已。<br/>
不過，誰又會在乎呢？<br/>
管羊吃的是什麼草料，反正確實變胖了，不是嗎？<br/>
那怕那是中毒的胖、是虛胖、是水腫腎臟壞掉的胖也沒關係，<br/>
反正，連什麼是毒不是毒，牧師自己也常不清楚。<br/>
比起紮實辛苦、按部就班，還不如找些速效的方法還比較實際一些，不是嗎？<br/>
<br/>
<br/>
其實，或許教會增長的方向都走偏了。<br/>
其實，教會要復興，要難也不難………………給我們一個好牧人就夠了…………一個願意為羊捨命的好牧人，一個願意尋找迷失小羊的好牧人。<br/>
這個牧人，他疼愛每隻小羊，視為自己寶貝；<br/>
他會為小羊的痛苦而悲傷流淚，小羊的痛就是他的痛，他會努力醫治小羊。<br/>
教會的大小，待遇的好壞，不是他在意的重點。<br/>
他所在意的，是希望小羊過得好。<br/>
<br/>
<br/>
很多牧師，可能從來也沒想過這個問題。<br/>
這問題，是何等尖銳，又何等令人心痛：<br/>
萬一有一天，教會付不出薪水，牧師還願意在這間教會牧會嗎？<br/>
<br/>
<br/>
你有多愛這間教會？<br/>
是嘴巴講的愛，還是真正從心底出來的愛？<br/>
有些假設問題，其實是很現實的試金石。<br/>
<br/>
<br/>
福音派神學大師卡森，曾分享過他父親的故事。<br/>
他的父親，是一位加拿大的牧師。<br/>
他的教會，位於福音很難傳的地區。<br/>
他的父親，學問不是多高，講道不是多厲害，<br/>
但是，他用全心的愛，來牧養那間小小的教會。<br/>
那間教會，會友只有幾十人（好像連五十人都不到），而且數十年看不見增長。<br/>
卡森自己說，他小時候常看見父親在房裡花很多的時間流淚為會友禱告。<br/>
有次，一個宣教團體來到那個地區，他們是很成功的團體，曾在許多國家與地區獲得很大的增長。<br/>
他們來到當地，很不屑的認為你們就是本事不好，所以教會才不能增長。<br/>
結果，他們在當地工作幾個月之後，放棄當地事工，離開了。因為，他們發現那套技術在當地是無效的，雖然很辛苦努力，但還是帶不出增長。<br/>
但是，卡森的父親，一個小教會的牧師，依然無怨無悔，盡他的本分，來牧養這樣的小教會。<br/>
直到過世，他父親還是沒親眼見到教會的增長。<br/>
然而，在他父親過世多年後，有次的佈道會，大家本來也沒多大預期，沒想到，竟然造成大爆滿，而且多人悔改信主。<br/>
當初，那個福音硬土地區的多位牧師們，奉獻一生、無怨無悔、不求掌聲的牧養、撒種、耕耘、把硬土鬆開、、、，終於在幾十年後，開始爆炸性增長，開花結果。<br/>
<br/>
<br/>
我們是不是能不再追求速效？<br/>
我們是不是能回歸古老牧會的年代？<br/>
就是很單純，用愛心來當牧羊人，不是靠嘴巴講，而是真正身體力行，認真探訪、認真照顧、關心會友？<br/>
先回歸好牧人這個很基本很基本的基礎上來，然後我們再來進行各種事工？<br/>
<br/>
<br/>
在過去的年代，許多牧師，真的是翻山越嶺、騎腳踏車半小時、騎老爺摩托車一小時，到各偏遠會友家裡去探訪、關心會友及其家庭、不論貧賤富貴、不論疾病與健康。<br/>
我敢講這句話，是因為我親眼見過許多這樣的老牧師，他們真的是這樣在牧養群羊的。<br/>
對這些牧師，我除了肅然起敬，還是肅然起敬，由衷的肅然起敬。<br/>
<br/>
<br/>
我是個癡人作夢的人，在這時代，講一個老掉牙的牧會型態。<br/>
但是，即使是老掉牙，我還是要講：<br/>
主啊，求你賜給我們這樣的牧者吧！<br/>
<br/>
<br/>
多給我們一些願意從最基本的當個疼愛小羊的好牧人開始，然後在這紮實的根基上認真傳福音的牧師；<br/>
而不是只想靠著靈恩、教會增長技術、成功神學、、、，用這種速成的方式來達成所謂「傳福音」神聖口號的牧師。<br/>
若真的回歸基本，我想，教會要不增長，恐怕都很難。<br/>
<br/>
<br/>
小小羊</p>
