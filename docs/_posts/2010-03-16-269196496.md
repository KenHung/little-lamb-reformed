---
title: 廢死刑？我這基督徒的看法（二）：新約已廢除死刑？
last_modified_at: 1547992595
category: 社會流行觀念的偏差
tags: ['死刑']
---

<p>在第一篇文章裡，我已經駁斥了矯正論基督徒的錯誤之處。<br/>
這篇文章，要繼續探討另一個問題：<br/>
死刑只在舊約，新約已經廢止？<br/>
<!--more-->這類的論述，還有很多，好比：<br/>
1. 主耶穌已經廢止摩西律法的刑罰，所以死刑也被廢止了。<br/>
2. 今天我們並沒有人照舊約摩西律法執行不守安息日就處死刑的規定，同樣的，摩西律法的死刑也都該被同理來廢止。<br/>
3. 主耶穌沒判行淫時被抓的婦女死刑（約8：1-11）。<br/>
4. 保羅對哥林多教會犯淫亂的人也沒判他死刑，只逐出教會而已（林前5章）。<br/>
5. 新約愛的主張，如：愛你的仇敵、不要以眼還眼，所以該廢除死刑。<br/>
6. 主耶穌被處死刑，代替全人類死在十字架上，所以之後所有死刑都該被廢止。<br/>
<br/>
<br/>
這麼多各式各樣的論述，其實核心都差不了多少，也就是雖然舊約裡死刑是無可否認，但『新約已經廢除死刑』。<br/>
這樣的論點，被大量廢死刑基督徒提出，以做為他們從聖經找出認為支持廢死刑的論據。<br/>
問題是，這樣的論述，站得住腳嗎？經得起檢驗嗎？<br/>
<br/>
<br/>
以下，就是我們對他們錯誤的駁斥：<br/>
<br/>
<br/>
1. 舊約的神權統治與新約的政教分離<br/>
基本上，舊約是一種宗教國家的模式，國家律法和舊約聖經是合在一起的，聖經法律就是國家法律。（但很不幸的，這樣的時間也不長）<br/>
但新約時代的主耶穌時期、使徒時期，都是政教分離：<br/>
政治受羅馬帝國管轄，信仰則是聖經。<br/>
羅馬帝國政府並未將聖經法律直接拿來當國家法律。<br/>
主張新約已經廢除死刑的基督徒，嚴重忽略了這個重要的背景因素。<br/>
換言之，在新約時期，死刑不再是由教會群體所負責，而是由國家政治權柄所負責，二者已經分割開來。<br/>
教會沒有審判或執行死刑（如：主耶穌沒有審判或執行死刑、保羅沒有審判或執行死刑），不表示新約就否定死刑。<br/>
事實上，這時死刑的權柄與判定，已經移轉到國家身上，並不是死刑就被廢止。<br/>
其實當時的教會與信徒，根本沒有審判死刑的權力，也沒有執行死刑的權力。<br/>
法利賽人恨透了主耶穌，想讓祂死，但祂們沒辦法透過宗教法庭處死祂，而是必須透過羅馬帝國的政府（巡撫彼拉多）來進行死刑的審判與執行。<br/>
也就是說，主耶穌以及保羅，在當時並沒有判人死刑的權柄，也沒有執行死刑的權柄。<br/>
當然，主耶穌並不打算判刑淫婦人死刑，保羅並沒打算判教會亂倫信徒死刑，<br/>
但是，這並不表示他們就反對政府判人死刑。<br/>
<br/>
<br/>
其實，『該撒的歸該撒，上帝的歸上帝』（太22：21），這樣的原則與分寸，才是新約時期的原則與寫照。<br/>
當時的政府並不判淫婦死刑，不判亂倫者死刑，所以基督徒服從政府，頂多只進行教會懲戒將犯罪者逐出教會，並不判人死刑。<br/>
但我們不能因此就反過來說新約已經否定死刑。<br/>
<br/>
<br/>
<br/>
<br/>
2.主耶穌自己被判死刑，祂並沒反對或反抗死刑<br/>
當主耶穌被羅馬政府彼拉多巡撫審判時，彼拉多警告主耶穌他具有生殺大權，可以判祂死刑。<br/>
請注意主耶穌的回答：<br/>
『若不是從上頭賜給你的，你就毫無權柄辦我』（約19：11）<br/>
主耶穌並沒反對死刑，或是反抗死刑，事實上，祂是認定政府具有死刑的權柄，並且這權柄可及於祂這上帝的兒子身上，而這種權柄是來自上帝的。<br/>
假使主耶穌認為新約時代就可以否定死刑，或是祂認為死刑罪大惡極必須廢除，當祂面對這種切身之事時，祂應該會做出廢止死刑的教導才對，而不是講出這種順服死刑的話。<br/>
我們都必須小心：<br/>
假使連貴為上帝的兒子，都服在權柄之下，並未反對或反抗死刑，<br/>
那我們這些跟隨主耶穌的基督徒，如何能用主耶穌自己來當作廢除死刑的支持者呢？<br/>
<br/>
<br/>
<br/>
<br/>
3.保羅不僅沒反對死刑，還講若犯該死的罪，他願意被處死<br/>
保羅被猶太人控告，被巡撫非斯都大人審判時，他自己講：<br/>
『我若行了不義的事，犯了甚麼該死的罪，就是死，我也不辭』（徒25：11）<br/>
換言之，保羅認為，假使他若犯了什麼大罪，觸犯了羅馬帝國的死罪，被政府判處死刑，他也心甘情願。<br/>
當然，他並沒有犯什麼死罪，但問題是，這句話非常清楚顯示出保羅並沒有否定死刑，也沒有反對死刑，而且還贊成用死刑來處死犯該死的罪的人，並且認為即使是他自己犯死罪被人判處死刑，他也贊成。<br/>
<br/>
<br/>
<br/>
<br/>
4.政府刑罰作惡者，是上帝的用人<br/>
『作官的原不是叫行善的懼怕，乃是叫作惡的懼怕。你願意不懼怕掌權的嗎？你只要行善，就可得他的稱讚；因為他是神的用人，是與你有益的。你若作惡，卻當懼怕；因為他不是空空的佩劍，他是神的用人，是伸冤的，刑罰那作惡的。所以你們必須順服，不但是因為刑罰，也是因為良心』（羅13：3-5）<br/>
這段經文很清楚告訴我們：<br/>
政府是上帝的用人，具有申冤、刑罰惡人的功能。<br/>
而且這權柄乃是來自上帝所賦予。<br/>
『在上有權柄的，人人當順服他，因為沒有權柄不是出於神的。凡掌權的都是神所命的』（羅13：1）<br/>
<br/>
<br/>
請注意這段經文裡的『佩劍』。<br/>
當然，這是一種比喻，用來表示『權柄』。<br/>
但問題是，『劍』，是在幹嘛用的？<br/>
是在殺人的，不是用來切菜的！<br/>
在全段經文裡，『劍』和前後文的『懲罰作惡』連在一起看，我們有沒有發現那種支持死刑的間接關連？<br/>
事實上，當時羅馬帝國是非常清楚有死刑這種懲罰的，<br/>
保羅這段經文，就是給羅馬帝國統治下的信徒看的，這表示他要信徒必須服從羅馬帝國對惡人懲處的律法，當然，這包括死刑。<br/>
沒錯，對於政府的權柄，基督徒必須優先順服神，<br/>
但問題是，保羅對於羅馬政府的死刑，並沒有意見，全本聖經對於死刑也沒有認為這種刑罰就不合聖經，相反的，不管新舊約聖經，都有死刑的處罰。<br/>
所以，聖經並沒教導基督徒必須反對懲罰惡人的死刑，反而要基督徒必須順服國家所定懲罰惡人的死刑的刑罰。<br/>
<br/>
<br/>
<br/>
<br/>
5.愛並沒有廢除公義<br/>
事實上，搬出『愛』這口號，就認為可以排除其他聖經裡公義與刑罰的經文，是很嚴重的斷章取義解經。<br/>
這部分，我們第一篇文章已經講過，不再重複。<br/>
別的不講：<br/>
上帝對犯罪者進行死刑擊殺，難道就是沒有愛心？<br/>
上帝因為有愛心，所以就不能判人下地獄受靈魂的死刑？<br/>
<br/>
<br/>
<br/>
<br/>
6.靈魂的赦免並沒有排除肉體的刑罰<br/>
信耶穌當然可以罪蒙赦免，<br/>
但問題是，這個赦免，並不是說信耶穌就不用受肉體的刑罰。<br/>
得赦免的，是『靈魂』！<br/>
肉體，並不能因此就免除。<br/>
否則的話，人人信耶穌，為何人人又都一樣會肉體死亡？<br/>
豈不該肉體與靈魂一起活著上天堂嗎？<br/>
大衛犯姦淫罪被赦免，但他的肉體刑罰依然存在，『刀劍必不離他的家』（撒下12：10）。<br/>
信了耶穌，你欠人家錢的罪被主赦免，但你一樣要還錢，不因你靈魂得赦免，你就不用還錢。<br/>
信了耶穌，你喝酒撞傷人的罪被主赦免，但你一樣要賠償，不因你靈魂得赦免，你就不用賠。<br/>
因此，企圖用信耶穌得赦免來否定死刑，是嚴重對赦免教義的誤解。<br/>
<br/>
<br/>
<br/>
<br/>
7. 舊約死刑在新約全部廢止？<br/>
舊約的死刑，確實有很多，在新約時代都已經被停止不用，這一點，我們絕對同意。<br/>
像：不守安息日者要處死，在新約時代，已經不再使用。<br/>
但問題是，因此就更進一步推論，變成新約時代已經『全面』停止死刑，或是反對死刑，那又走過頭，越過聖經了。<br/>
前面好幾個例子，已經讓我們看見，新約時代，主耶穌、保羅，依然沒有廢止死刑或反對死刑的任何教導，反而是承認死刑的合法性。<br/>
事實上，我們該作的，是小心檢討舊約聖經裡死刑的規定，<br/>
將一些有特定時代背景，或僅限於舊約時代是用的死刑規定停用，<br/>
但對一些跨時代都一樣適用的舊約死刑規定，依然必須保留，而不是全面廢止，這樣才對。<br/>
<br/>
<br/>
舊約在新約時代的使用，是一個很複雜的議題，無法三言兩語講清楚。<br/>
舊約很多東西確實在新約時代被廢止，但假使認為『全面』廢止，這樣的解經就太離譜了。<br/>
<br/>
<br/>
<br/>
<br/>
從以上這些論點，我們應該可以很清楚發現，<br/>
新約並未廢止死刑，也沒有反對死刑，或是認為死刑不合理。<br/>
企圖用『新約已廢止死刑』來做為論述要點，是不合理的，經不起檢驗的。<br/>
<br/>
<br/>
<br/>
<br/>
下一篇，我們要繼續來看主張廢死刑基督徒的另一個重要論點：人權（生命權）。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
小小羊</p>
