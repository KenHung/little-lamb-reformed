<p>人權、生命權，這是另一個很常見的反死刑基督徒的論點。<br/>
這種看法主要是認為：<br/>
人是照上帝形像造的，人是尊貴的，不能殺人（包括判人死刑），不能以暴制暴。<br/>
<!--more-->但是，這樣的論點，真的站得住腳，經得起聖經檢驗嗎？<br/>
<br/>
<br/>
我們駁斥的主要要點是：<br/>
1. 要講人權、生命權，不能只講『加害者』，而必須同時考慮『受害者』。<br/>
2. 『生命權不可剝奪』這個觀念，在聖經裡，不是用來為『加害者辯護』（廢除死刑），而是用來對『受害者申冤』（贊成死刑）的。<br/>
3. 申冤、報應、刑罰，是聖經裡公義的原則，而且沒有被廢棄，也是末世審判的原則。<br/>
<br/>
<br/>
我們絕對同意，正因為人是尊貴的，在一切動物裡，只有人是照上帝形像所造的，<br/>
因此，這是人類人權與生命權很重要的基礎，任何人不得隨意剝奪。<br/>
但是，要探討這個角度，就不能只單挑『加害者』來講，而迴避『受害者』的部分。<br/>
人的生命是尊貴的，這一個條件，不僅對加害者成立，對受害者也一樣成立。<br/>
事實上，正因為人人的生命都是尊貴的，不分貧富貴賤、男女老幼，<br/>
因此，人人的生命都該受到尊重。<br/>
既然如此，當任何人殺害另一個人的生命時，表示加害者違反另一人的生命權，不尊重另一人的生命權，這樣，更是大罪。<br/>
而對這種大罪的懲處，就是『以命還命』。<br/>
<br/>
<br/>
正因為人類生命權是何等神聖與尊貴，所以上帝親自對挪亞講：<br/>
<br/>
<br/>
『流你們血、害你們命的，無論是獸是人，我必討他的罪，就是向各人的弟兄也是如此。凡流人血的，他的血也必被人所流，因為神造人是照自己的形像造的』（創9：5-6）<br/>
<br/>
<br/>
假使要說殺人者死刑的規定，那麼，這個創世記經文，就是強而有力上帝為人類設立的重大依據。<br/>
正因為人是照上帝形像造的，所以，殺害任何人的，不管是人，是動物，都是大罪。<br/>
而正因為人是照上帝形像造的，所以，殺人的人，他也要被殺。<br/>
<br/>
<br/>
請注意，哪些罪可以判死刑，這是另一個議題，我們不多涉入。<br/>
但問題是，聖經裡對於殺人罪可以判死刑，並沒有什麼爭議，<br/>
而且，上帝親自做出了殺人罪判死刑的規定，還要人去執行。<br/>
『凡流人血的，他的血也必被人所流』<br/>
這經文告訴我們：<br/>
1. 殺人的，也要被殺。<br/>
2. 加害者的死刑，由人來執行。（被人所流）<br/>
<br/>
<br/>
接下來，是上帝頒佈給摩西的十誡。<br/>
十誡，是全人類都必須遵守的法則，不僅限於以色列人而已。<br/>
十誡第六誡，就是『不可殺人』。<br/>
接下來的摩西五經與殺人有關更詳細的解釋與刑罰，都是在針對這條誡命做出更進一步的解釋。<br/>
如：<br/>
民數記35章，就詳細解釋與規定：<br/>
1. 『誤殺人的』，可以躲到逃城，在裡面不會被殺。<br/>
2. 『故殺人的』，一律死刑（即使躲到逃城也一樣）。<br/>
<br/>
<br/>
不僅如此，聖經還對此提出『施展公義才能潔淨土地』的觀念：<br/>
『若有在地上流人血的，非流那殺人者的血，那地就不得潔淨』（民35：33）<br/>
看見上帝對那種公義伸展的潔淨觀念了嗎？<br/>
對邪惡殺人犯不處死刑，是讓那塊地不潔淨！<br/>
而上帝嚴格要求所有的人必須潔淨（不僅限於基督徒而已，而是普世性的），否則會讓上帝痛恨與厭惡！<br/>
當然，這裡的『土地』，不僅僅只是說我們腳下所踏的土地這種字面意思，而是可以擴大用於社會、國家意義的。<br/>
也就是說，殺人的罪，會污染整個社會、國家，使之不潔淨，遭致上帝發怒而刑罰，<br/>
而刑罰殺人者，才能使社會、國家獲得潔淨，避免全體被上帝刑罰。<br/>
<br/>
<br/>
『在這一切的事上，你們都不可玷污自己；因為我在你們面前所逐出的列邦，在這一切的事上玷污了自己；連地也玷污了，所以我追討那地的罪孽，那地也吐出它的居民。故此，你們要守我的律例典章。這一切可憎惡的事，無論是本地人，是寄居在你們中間的外人，都不可行，在你們以先居住那地的人行了這一切可憎惡的事，地就玷污了，免得你們玷污那地的時候，地就把你們吐出，像吐出在你們以先的國民一樣』（利18：24-28）<br/>
<br/>
<br/>
『故殺人、犯死罪的，你們不可收贖價代替他的命，他必被治死。、、、這樣，你們就不污穢所住之地，因為血是污穢地的；若有在地上流人血的，非流那殺人者的血，那地就不得潔淨。你們不可玷污所住之地』（民35：31-34）<br/>
<br/>
<br/>
很多人根本就沒搞清楚上帝對『潔淨』的看重。<br/>
聖經裡的潔淨，有很多含意，像：分別為聖、聖潔、、、。<br/>
有些罪，上帝會寬容，會容忍人，以便等候人悔改。<br/>
但也有些罪，是嚴重觸犯上帝潔淨觀念的，特別是那種會『使地被玷污』的。<br/>
這種會玷污地的罪，主要是像：亂倫、同性戀、殺人、、、。<br/>
這種罪假使在一塊地方大量流行，那麼，這種罪的大惡程度，是會讓上帝施展嚴厲的刑罰，『使地吐出』這些土地的人民（全面被滅）。<br/>
這代表什麼，知道嗎？<br/>
這表示：<br/>
假使你們對邪惡殺人者不處死刑，那麼，連你們這些沒殺人的居民，也一樣會被牽連，遭地吐出來！因為，你們住的那塊地，已經被玷污了！<br/>
所多瑪蛾摩拉（上帝自己動手）、迦南七族（上帝要以色列人動手），只因使地被玷污，結果不分男女老幼，不分自己有無觸犯，全面被擊殺的慘狀，難道不足以使我們心生畏懼而警惕？<br/>
<br/>
<br/>
當然，我們認為聖經支持死刑，並不是單靠這一些經文，就可以成立。<br/>
但是，『殺人者死』、『以命還命』這樣的觀念與刑罰，在聖經裡不斷反覆出現，連聖經最後一卷的啟示錄，都還在出現，卻也是不能否認的事實。<br/>
<br/>
<br/>
『擄掠人的，必被擄掠；用刀殺人的，必被刀殺。聖徒的忍耐和信心就是在此』（啟13：10）<br/>
<br/>
<br/>
而這個觀念，事實上，就是因為看重『每個人』的生命權。<br/>
因此，若有受害者生命權被人惡意剝奪，為維護受害者生命權的尊貴，所以用剝奪加害者的生命權做為懲罰，是聖經的教導。<br/>
正因為生命權是如此尊貴，所以也只能用生命權來刑罰，不可用錢來代替，用罰款來取代死刑的懲處。<br/>
<br/>
<br/>
『故殺人、犯死罪的，你們不可收贖價代替他的命，他必被治死』（民35：31）<br/>
<br/>
<br/>
換言之，當基督徒用『生命權不可剝奪』來為『加害者』辯護，進而否定死刑時，<br/>
聖經更清楚的『生命權不可剝奪』的解法，反而是針對『受害者』來講，也就是加害者無權剝奪受害者的生命權，而他們殘酷這樣殺人，就必須這樣被處死。<br/>
<br/>
<br/>
我再講一次：<br/>
『生命權不可剝奪』這條鐵律，在聖經裡，不是用來為『加害者辯護』（廢除死刑），而是用來對『受害者申冤』（贊成死刑）的。<br/>
拿這條規定來為加害者辯護，藉此否定死刑的基督徒，是嚴重扭曲了這種觀念的意思與用途的。<br/>
<br/>
<br/>
<br/>
<br/>
在此，也稍微提一下很多主張死刑者講的『安慰家屬』的觀念。<br/>
很不幸的，這個論點，即使有，也不是聖經裡死刑主張的主要論點。<br/>
身為基督徒，我們凡事要回到聖經來。<br/>
就算我們支持死刑，也必須依照聖經的根據來支持，而不是拿沒聖經依據的東西來支持。<br/>
沒聖經依據的東西，我們不要拿來用。<br/>
上帝有講的，上帝自己背書，我們可以理直氣壯勇敢的講；<br/>
上帝沒講的，上帝不背書，我們不要隨便拿來，以免自取其辱。<br/>
比方說：<br/>
判加害者死刑以安慰家屬這個論點，輕易就能被廢死刑者駁斥，他們會說：<br/>
那些全家都被殺了，沒有留下任一親屬的，判加害者死刑，是安慰了哪一個親屬啊？<br/>
看見了嗎？<br/>
沒上帝背書的東西，你隨便跟群眾流行走，人家講你也拿來講，會自取其辱的喔！<br/>
<br/>
<br/>
反而聖經裡講的，不是什麼『安慰家屬』，而是『安慰死者』。<br/>
為什麼？<br/>
很簡單：<br/>
人的肉體死了，但靈魂依然存在。<br/>
公義的施展，是因為這死者的靈魂在呼喊，要求申冤，而上帝施展公義來安慰死者靈魂：<br/>
<br/>
<br/>
『你兄弟的血有聲音從地裡向我哀告』（創4：10）<br/>
<br/>
<br/>
『我看見在祭壇底下，有為神的道、並為作見證被殺之人的靈魂，大聲喊著說：聖潔真實的主啊，你不審判住在地上的人，給我們伸流血的冤，要等到幾時呢？於是有白衣賜給他們各人；又有話對他們說，還要安息片時，等著一同作僕人的和他們的弟兄也像他們被殺，滿足了數目』（啟6：9-11）<br/>
<br/>
<br/>
『他的判斷是真實公義的；因他判斷了那用淫行敗壞世界的大淫婦，並且向淫婦討流僕人血的罪，給他們伸冤』（啟19：2）<br/>
<br/>
<br/>
至於處加害者死刑，<br/>
可以由『上帝自己』動手，來為受害者申冤；<br/>
也可以由『上帝託人』來動手，來為受害者申冤。<br/>
前者，經文太多了，不再列舉。<br/>
後者，我舉出一段，是關於上帝膏立耶戶當國王，然後要他執行死刑，殺盡亞哈王全家，來為受害者申冤。<br/>
並且上帝對耶戶的執行非常滿意，還因此祝福他可以子孫當國王四代。<br/>
更重要的是，上帝說這種判人死刑、執行死刑，是上帝眼中『看為正的事』。<br/>
這種上帝挑人當劊子手，還大大獎勵，並說這樣是上帝看為正的事的經文，我不知那些主張廢死刑的基督徒，要如何圓解？<br/>
<br/>
<br/>
『耶戶就起來，進了屋子，少年人將膏油倒在他頭上，對他說：耶和華以色列的神如此說：我膏你作耶和華民以色列的王。你要擊殺你主人亞哈的全家，我好在耶洗別身上伸我僕人眾先知和耶和華一切僕人流血的冤』（王下9：7）<br/>
<br/>
<br/>
『耶和華對耶戶說：因你辦好我眼中看為正的事，照我的心意待亞哈家，你的子孫必接續你坐以色列的國位，直到四代』（王下10：30）<br/>
<br/>
<br/>
這種經文，用什麼去除公義的愛心、加害者人權、加害者生命權、不可殺人是包括不可殺加害者、、、、之類的論述，根本無法合理解通的。<br/>
這種經文，本來就是唯有按照正意來解經，也就是照公義、申冤、報應、刑罰、殺人者死、、、的總原則來解經，才有辦法合理解通的。<br/>
事實上，當我們不斷將『上帝是公義的』這樣的屬性去除時，我們根本就是離聖經真理越來越遠，而講出看似好像有愛心但其實根本不是真正合乎聖經的愛心的錯誤道理來。<br/>
<br/>
<br/>
其實，在聖經裡，我們看見上帝自己常常就是死刑的審判者與執行者。<br/>
而對於主張廢除死刑的基督徒而言，以下這問題，會是一個致命性的難題：<br/>
上帝擊殺人的時候，以你們的理論，上帝合乎你們要求的罪犯的人權、生命權嗎？<br/>
上帝擊殺亞倫兩個兒子、上帝擊殺烏撒、上帝擊殺亞拿尼亞撒非拉、、、、。<br/>
<br/>
<br/>
當然，我們可以說這些被上帝擊殺的人是『確定犯』，不是『嫌疑犯』；<br/>
我們也可以說人類不是上帝，面對罪犯常常只是『嫌疑犯』，無法確定是『確定犯』。<br/>
這樣的理由，當然很好，也確實是死刑議題很重要的部分。<br/>
但問題是，這依然沒有回答我的問題，<br/>
因為，上帝本來就可以託人執行與審判死刑，<br/>
而且也不是每次都是很難判斷、很具爭議的嫌疑犯而非確定犯。<br/>
亞哈王及王后耶洗別的罪行，是非常清楚的；<br/>
那些被上帝直接擊殺的，也都是很確定清楚的，旁人知道的。<br/>
確實，有些死刑犯是否有犯罪，是大有爭議的。<br/>
但也不能否認，有些死刑犯，是不管他們自己，還是別人，大家都同意確實有觸犯嚴重罪行，而且沒有爭議的。<br/>
上帝對這些人處死刑，難道上帝就違反人權生命權？<br/>
<br/>
<br/>
事實上，總括以上論述，企圖以人權、生命權之類理由來否定死刑，在聖經的檢驗下，是站不住腳的。<br/>
<br/>
<br/>
當然，我們不是說為受害者申冤，就可以動輒處加害者死刑、就可以殘酷行刑、可以不公義對待嫌疑犯、可以凌虐嫌疑犯、可以凌虐犯人、可以隨意輒處死刑、可以不在乎程序正義、、、、，不是這樣的。<br/>
事實上，很多主張廢死刑的基督徒所提出的死刑偏差，確實是很重要的提醒，也是每位基督徒必須認真思考的。<br/>
但是，這是一種對過去長期不當死刑的偏差的反動，原本是一種值得稱許的好意，但卻嚴重越過聖經，走到另一個極端去。<br/>
本來是好好改善死刑的一切相關問題，結果卻變成主張極端，認為不廢死刑不可。<br/>
在教會歷史上，這樣的事情，也屢見不鮮。<br/>
靈恩運動，就是很明顯的例子。<br/>
原本只是對於傳統教會偏差的不滿與糾正，結果卻走到極端，往越過聖經，更嚴重偏離聖經的路上狂奔。<br/>
<br/>
<br/>
然而，身為基督徒，要如何面對各式各樣的理論學說與正反意見？<br/>
很簡單，我們依然持守唯獨聖經，凡事用聖經來檢驗的基本原則，仔細分辨是非對錯、何者合聖經何者不合聖經便是。<br/>
而在廢不廢死刑的議題上，很不幸的，到目前為止，主張廢死刑的基督徒的論點，在我個人來看，實在無法有力說服我。<br/>
他們的論點都經不起我用聖經來檢驗，而他們提出的聖經依據都站不住腳。<br/>
除非有基督徒能提出更強而有力的聖經依據，也能合理解通聖經裡大量公義與報應的經文，否則的話，我目前只能說：<br/>
死刑是合乎聖經的！<br/>
<br/>
<br/>
<br/>
<br/>
下篇文章，是針對比較一般性的反對死刑的理由，來做出駁斥。<br/>
雖然這些理由比較沒有像前面這三篇一樣，是直接拿出聖經觀念來論述，<br/>
但因為那些理由也是主張廢死刑的基督徒所常常拿出來的一般性理由，<br/>
所以我們一樣用聖經來檢驗那些理由，看能否通過聖經檢驗。<br/>
<br/>
<br/>
