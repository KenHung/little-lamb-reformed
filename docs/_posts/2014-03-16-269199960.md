<p>常有弟兄姊妹拿反對者的問題來問我，希望我能幫忙回答。<br/>有些，我會簡單回答；<br/>但更多的時候，我只能說-----我愛莫能助。<br/>以下是一封我的回信。<br/>裡面對我的做法做出說明，也對一些反對者的問題解答做出思考上的示範，<br/>提供大家參考。<br/><!--more-->~~~~~~~~~~~~~~~~~<br/>基本上，我已經很久不願意和這種反基督教的人在網路上進行對話討論了。<br/>在時間有限的情形下，我個人領受主要是針對主內、慕道友或對基督教沒什麼敵意的未信者。<br/>至於和反基督教的網友在網路上進行辯論，成效其實不彰（除非，你是打算透過網路上公開的辯論內容，給其他『隱性讀者』更清楚基督教）。<br/> <br/><br/>你所提供的這類反對者言論，是很標準反對者對基督教攻擊的模式：<br/>從『遺棄論』、『上帝不慈愛』出發。<br/> <br/><br/>這個老招數，不僅是反基督教人士愛用，<br/>連當初多特會議時，反對加爾文主義的阿民念主義者，也一樣用這種老招數。<br/>他們論述的重點與思路，其實很簡單：<br/>1.上帝遺棄人、上帝刑罰人、上帝容許世上有悲慘的苦難<br/>2.所以，可以由此「證明」上帝不慈愛<br/>3.更進一步可「證明」聖經所講的是假<br/> <br/><br/>其實，世上有苦難，並無法證明上帝是慈愛，還是上帝不慈愛。<br/>正如科學不能證明有上帝，也不能證明無上帝，<br/>科學面對『有沒有上帝』這種問題，只能謙卑的說『我不知道』。<br/>因此，世上有苦難，和宇宙有上帝，單單這二者，並無法讓我們做出上帝慈愛或不慈愛的結論。<br/>所以，不要被他們唬了。<br/> <br/><br/>我換個方式描述好了：<br/>家裡有父母，小孩受重傷。<br/>我們並無法因此就說父母虐待小孩，或是父母沒虐待小孩。<br/>要『證明』，還要有更多證據才行。<br/>單單靠『孩子受重傷』的『現象』，就要來『證明』父母虐待，未免太粗糙了吧？<br/> <br/><br/>請留意當時多特會議眾多堅守真理的先聖先賢，他們所堅守的真理是什麼？<br/>他們堅守的，就是『TULIP』（加爾文主義五要點）：<br/>1.全然的墮落<br/>2.無條件的揀選<br/>3.限定的救贖<br/>4不可抗拒的恩典<br/>5.聖徒永蒙保守<br/> <br/><br/>再請注意其中第一點，也就是我們常忽略的，是什麼？<br/>是『全然的墮落』！<br/>什麼是全然的墮落？<br/>就是全人類都被罪徹底污染了，沒有任何一吋地方是乾淨的了，<br/>所有的良心、理性、道德、、、通通都被污染了，人的靈性已經死在過犯之中了。<br/>換另一種方式來講，就是全人類都有罪，都注定要下地獄，沒有人有資格上天堂。<br/>更簡單講，就是『原罪論』。<br/> <br/><br/>看見沒？<br/>反對者不肯先面對『人有罪』的重要聖經基礎，<br/>然後避開在這基礎上發展出來的上帝慈愛與上帝刑罰，<br/>之後就直接訴諸『上帝刑罰所謂「無辜者」』的論述，<br/>最後得出『上帝不公平』的結論。<br/>看見反對者這種論述錯誤的地方了嗎？<br/> <br/><br/>我換個方式來講：<br/>1.『假使人有罪』---------那麼，上帝本來就沒有義務要救人。<br/>人因為自己的罪下地獄，是罪有應得，不是上帝推人下地獄，是人自己犯罪而下地獄的。<br/>這時，假使上帝有救人，這樣的上帝，豈不是慈愛嗎？<br/><br/>2.『假使人無罪』-------那麼，上帝讓無罪的人下地獄，當然不慈愛。<br/>上帝讓無罪的人受苦難，當然不慈愛。<br/> <br/><br/>問題是，人是無罪者嗎？<br/> <br/><br/>一個人自己去強姦殺人放火，後來被抓起來判死刑，然後他有資格說『法官不慈愛，讓我受死刑的苦難』嗎？<br/> <br/><br/>反對者還會因此狡辯：<br/>可是，嬰兒沒犯罪。<br/> <br/><br/>問題是，聖經早就明講了：<br/>那請解釋為何沒犯罪的嬰兒會死亡呢？<br/>這是所有宗教信仰（包括無神論），都必須面對的殘酷事實。<br/>為什麼所謂「無罪」的嬰兒會死亡？<br/> <br/><br/>佛教、印度教的說法很簡單：<br/>因為他們『前世』的罪，於今世償還，輪迴報應。<br/>基督教的說法也很簡單：<br/>因為他們有『原罪』，而罪的工價乃是死，所以嬰兒雖然無法犯出本罪，但一樣會因原罪而死。<br/>但雖然因原罪而死，這可不表示他們靈魂就會下地獄。<br/>事實上，多數神學家依照聖經推論，認為無本罪的嬰兒，死亡是上天堂。<br/>（註：『本罪』是指自由意志犯出的罪』）<br/><br/><br/>無神論者呢？<br/>根本就解答不出為何嬰兒會死、為何宇宙間有人類，<br/>甚至，根本無法解釋為何人要活著、為何人不死了算了、為何人不努力做壞事。<br/>其實，真誠的無神論者，都必須承認：<br/>假使沒有上帝，人生毫無意義，死了算了。<br/>那些還不想死的無神論，根本就是自欺欺人者。<br/> <br/><br/>上面講了一堆，重點很簡單：<br/>不要掉進反基督教者的陷阱。<br/>他們迴避『人有罪』（原罪論），我們就要逼他們重回這個聖經基本立場來，在這立場下，我們再來討論上帝的各種作為。<br/>要不然，我們絕對無法回答他們的問題。<br/>請千萬注意，他們問題的基本立場，就是『人無罪』（普救論）（無罪論），我們絕不能在他們這種立場下，和他們進行攻防。<br/>事實上，那些反對者引用聖經來反聖經，根本就是亂來。<br/>假使他們要引用聖經，就請他們尊重聖經裡『人有罪』的基本假設。<br/>如果他們不肯接受，那也請他們不需引用聖經，因為聖經變成無意義的書，毫無可信可言。這時，基督教的上帝慈不慈愛，都變成無法驗證、也變成不重要，既然如此，大家就不需浪費時間去討論上帝慈不慈愛。<br/>當基督徒宣稱聖經無誤，那就表示假使你不願意接受全本聖經，那我們就無法討論，因為你不能只接受聖經寫的某些部分，卻不接受聖經寫的另一些部分。<br/>假使你願意接受『全本』聖經無誤（這不表示你就要相信，但你必須與我先在公平的基準點上，來進行討論），那你就不能只引用你要引用那些，卻拒絕我要提出的這些經文。<br/> <br/> <br/><br/><br/>至於『為什麼這麼多人厭惡基督教』，這哪有什麼好講的？<br/>什麼教都有人會反對，不是嗎？<br/>反倒是應該請發問者自己講一講，為什麼他會厭惡基督教？<br/>要不然，人人原因都不同，講都講不完。<br/>有些人，搞不好只因為小時候被基督徒不小心讓他走路跌倒，他就從此懷恨在心，開始反基督教。<br/>要解釋這些，我們哪解釋得完？<br/>事實上，我很不客氣講：<br/>那些反基督教者，幾乎都是因為過去曾被某些基督徒的『行為』所傷害過，所以反過來反對基督教『教義』，也就是根本就是把『個人恩怨』擴大成『種族仇恨』。<br/>換言之，這些人根本就是爛人、不誠實的人。<br/> <br/> <br/> <br/><br/>邏輯證明耶穌不誠實部分，這表示發言者連基督教在信什麼都不知道，還敢講什麼邏輯？<br/>基督教講的是：<br/>我們當然做不到耶穌的要求，所以我們不能靠『做』來得救，只能靠『信』來得救。<br/>我們做不到，所以，我們用信的，因為主耶穌全部做到了，而我們只要信，耶穌就代替我們完成那些部分，使我們的罪被遮掩赦免。<br/>這是基督教很基本的論點，連這都不知道，講什麼邏輯啊？<br/> <br/><br/>我換個方式再講一次：<br/>世界的宗教是要人去『做』（好行為），才能得救，也就是『因行為稱義』。<br/>基督教卻截然不同，它講的是『信』，『信』才能得救，『做』絕對無法得救，因為沒有人做得到。<br/>要如何完成絕對無罪、絕對行出好行為而得救？<br/>就只能靠『信耶穌』。<br/>這論點，就是『因信稱義』。<br/> <br/><br/>其他宗教，是『行為得救』，靠好行為來得救。<br/>基督教是『因信得救』，信了之後，我們再來努力行出好行為。<br/>乍看之下，好像都是『勸人為善』所以都一樣，不是嗎？<br/>但事實上，差遠了！<br/>因為：<br/>其他宗教說的是，你要先努力做個好孩子，父母才願意接受你當他們兒子。<br/>假使你不乖，或是你以後不乖，你隨時會被父母丟掉。<br/>基督教說的是，我被父母生出，成為他們兒子，然後我才努力當乖小孩。<br/>即使我做不好，我一樣是父母的孩子，血緣關係永不改變。<br/>但這不是說父母不會管教我，事實上，父母會嚴厲管教，但絕不丟棄。<br/> <br/><br/>我要如何當父母的兒子？<br/>其他宗教說的是『靠好行為』才能變成兒子，<br/>基督教說的是『靠信耶穌』。<br/> <br/><br/>狗抓一萬隻老鼠（好行為），永遠也不會因此而變成貓（得救）。<br/>要先被DNA改造，換成貓的DNA（重生），這才是真正的貓。<br/>一旦變成真正的貓，自然就會有天然慾望要去抓老鼠，只是抓多抓少而已。<br/> <br/> <br/><br/><br/>以上所說，只是粗略介紹方向，並不是詳細攻防。<br/>但是，你假使覺得上帝感動你去進行對話，<br/>那麼，這些方向，或許可以供你參考。<br/> <br/><br/><br/><br/><br/><br/><br/>小小羊<br/><br/><br/><br/><br/><br/><br/><br/>
